## 📌 Project Overview

This project performs an Exploratory Data Analysis (EDA) on ConnectaTel customer data to identify customer usage patterns, improve data quality, and generate actionable business insights.

The analysis includes data cleaning, missing value treatment, customer segmentation, outlier detection, statistical summaries, and data visualization to better understand customer behavior and support business decision-making.

---

## 🎯 Objectives

- Assess and improve the quality of the datasets.
- Identify missing values, invalid data, and outliers.
- Analyze customer demographics and service usage.
- Segment customers based on age and usage behavior.
- Generate business insights and recommendations for ConnectaTel.

## 🛠️ Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📈 Analysis Workflow

### 1. Data Cleaning
- Missing value analysis
- Invalid value detection
- Sentinel replacement
- Date formatting
- Missing data mechanism (MAR) evaluation

### 2. Exploratory Data Analysis (EDA)
- Statistical summaries
- Distribution analysis
- Histograms
- Boxplots
- Outlier detection using the IQR method

### 3. Customer Segmentation
- Age groups
- Usage groups
- User profile creation through dataset merging

### 4. Data Visualization
- Histograms
- Count plots
- Boxplots

### 5. Business Insights
- Customer behavior analysis
- High-usage customer identification
- Business recommendations for new service plans

---

## 📌 Key Findings

- Most customers belong to the **Adult** age segment.
- Most users fall into the **Medium Usage** category.
- High-usage customers represent a valuable commercial opportunity.
- Outliers correspond to legitimate heavy users rather than data errors.
- Data quality issues were successfully identified and addressed before analysis.
